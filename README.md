# WrittenBookEditor

一个 Python 编写的 Minecraft 成书编辑器。

## 功能

- 实时编辑并浏览成书内容.
- 支持多种导出格式, 包括指令文本文件，mcfunction, 数据包.

## 注意事项

以 1.21.0 的字体系统为准，其他版本的 Minecraft 可能出现轻微错位问题.

在 Minecraft 中开启“强制使用 Unicode 字体”. 并在该软件中开启同样的设置可以解决大部分错位问题.
